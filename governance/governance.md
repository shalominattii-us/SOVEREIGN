# SOVEREIGN Governance Charter

## Core Principles
1. Self-Determination
2. Mesh Consensus
3. Transparency
4. Resilience

## Identity Framework
- DID per node
- X.509 attestation

## Consensus Rules
- 2/3 majority for changes
- PoS weighted voting
- Emergency: 5-of-9 multisig
